Cool Poll adds polling functionality to your website.Very easy to be used.



== Description ==

Create polls with an easy way. With Cool Poll plugin you can create a poll and add it to a post or page so users can vote. You can create also unlimited polls with as much as 10 answers each.



== Installation ==

1. Download and unzip
2. Upload to your `wp-content/plugins` folder
3. Activate from the dashboard
4. Go to the new Polls menu page
